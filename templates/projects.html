StepMania Unlock Code Generator

Developed during: 2006 - 2008
Developed using: Python, PyGTK

Graphical program for generating unlock codes for the StepMania dance simulation game. This allows for locking of songs in an arcade style system.

This program was originally written in Python but a Java version was also made. The Python releases are the only ones to have a supported public release.

Doom-PSP S.S. (Shoulder Strafe)

Developed during: 2005
Developed using: C

A port of Doom for the PlayStation Portable with many added features compared to the original port. Features include: overclocking, custom button configuration and WAD loader. This version was later adopted by StereoMike.
